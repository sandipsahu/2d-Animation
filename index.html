How it works

Quick guide to using ScriptForge:

1. API Key — Click ⚙ API Key in sidebar and paste your free Gemini key (one-time setup).
2. Create Project — Click + New, give a name, paste master prompt. Make as many as you need.
3. Chat — Tap a project's name area to open it, then type in the message box at the bottom.
4. Edit / Delete — Each project has its own Edit and Delete buttons below it. Nothing deletes without confirming.
5. New Chat — Top bar button clears current chat only. Master prompt stays safe.
All data saved in your browser only. Keep master prompt files backed up.
