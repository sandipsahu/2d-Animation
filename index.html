How it works

1. Set API key (once) — click ⚙ API Key in the sidebar and paste your free Gemini key.

2. Create a project — click + New, give it a name, and paste your master prompt. You can make as many projects as you want (Jagannath, Cinematic, History, etc.)

3. Chat freely — select a project and start chatting. The master prompt guides everything. Paste scripts, say "haan aage badho", ask for changes — full conversation like ChatGPT.

4. New Chat — click "New Chat" in the top bar to start fresh (master prompt stays, only chat resets).

All data saved in your browser only. Clearing browser data will erase projects — keep master prompts backed up.
